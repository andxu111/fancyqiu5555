## 📊 练习内容概览

本项目包含5个核心练习，涵盖了Matplotlib子图和布局的主要技术：

### 🔹 练习1：plt.subplot() 基础
- **内容**: 创建上下两个子图展示智能手表和蓝牙耳机销量
- **技术点**: 基础子图创建、图表样式设置
- **图表类型**: 柱状图、折线图

### 🔹 练习2：plt.subplots() 与坐标轴共享
- **内容**: 共享Y轴的左右子图对比社交平台用户性别分布
- **技术点**: 坐标轴共享、水平柱状图
- **图表类型**: 水平柱状图

### 🔹 练习3：plt.subplot2grid() 非对称布局
- **内容**: 2x2非对称布局展示网站流量分析
- **技术点**: 不规则网格布局、多图表组合
- **图表类型**: 饼图、柱状图

### 🔹 练习4：ax.twinx() 双坐标轴
- **内容**: 股票价格与成交量双坐标轴图表
- **技术点**: 双Y轴、混合图表类型
- **图表类型**: 柱状图+折线图组合

### 🔹 练习5：GridSpec 自定义布局
- **内容**: 大学招生数据的复杂自定义布局
- **技术点**: GridSpec高级布局、多图表协调
- **图表类型**: 折线图、饼图、柱状图
